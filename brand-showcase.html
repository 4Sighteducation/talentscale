Usage Guide

Primary Palette

Jade #059669 leads the brand. Mint #34d399 is the light gradient stop on dark backgrounds. Always use the gradient for the mountain icon.

Accent Colour

Burnt Orange #C7632A is the accent exclusively. Use it for "SCALE", base lines, CTA buttons, and active UI states. Restraint is the whole point.

Dark Mode First

The brand reads strongest on dark. logo-dark.svg is the primary lockup. Use logo-light.svg for print and light interfaces.

Minimum Sizes

Wordmark: 160px wide minimum. Icon mark: 24px minimum in product. The 16px favicon is for browser tabs only.

Social Images

Export og-image.svg as PNG at 1200x630. Export twitter-card.svg at 1200x600 for X/Twitter large card format.

App Icons

Use the 256px rounded variant for Apple Touch. Submit at 1024x1024 PNG for App Store. Use dark-bg square for PWA manifests.
